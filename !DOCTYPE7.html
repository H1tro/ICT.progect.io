Technology Glossary

ROM (Read-Only Memory): A type of internal memory on a chip which is permanent/non-volatile and cannot have its contents changed by the user. It is used to hold the boot routines used to start the computer when the power is switched on.

Router: A hardware device that can decide the path an individual packet of data should take to arrive in the shortest possible time. It enables several computers to share the same connection to the internet.

RSI (Repetitive Strain Injury): A painful muscular condition caused by repeatedly using certain muscles in the same way.

RTF (Rich Text Format): A file format that saves text with a limited amount of formatting. Rich Text Format files use the file extension .rtf.

Queue: A collection of data items waiting to be processed or transmitted.

Sensors: Devices that measure physical quantities such as temperature, pressure, humidity, etc.

Serial/Sequential Access: Data is accessed from the storage media by starting at the beginning and continuing until the required data is found. It is a very slow form of access, typically used with magnetic tape.

Serif: A small decorative line added to the basic form of a character (letter, number, punctuation mark, etc.).

Smartphone: A mobile phone with an advanced operating system that combines the features of a mobile phone with other mobile devices, such as web browsers, media players, Wi-Fi, cameras, GPS navigation, etc.

Smishing: Using text messaging to send fraudulent messages that try to steal your credit card, banking, or other personal details.

SMS (Short Message Service): A service that provides texting.

Rules Base: One of four parts of an expert system, made up of a series of IF, AND, THEN statements that closely follow human-like reasoning.

Sans Serif: A set of typefaces or fonts that do not use the small lines at the end of characters, known as serifs.

Scanner: An input device that captures images, useful for digitizing old non-digital photographs, paper documents, or pictures in books.

Screenshot: A copy of what is seen on a computer screen. It can be obtained by pressing the Prt Scrn button, then the copy of the screen is placed in the clipboard, which can then be pasted.

Search Engine: A program that searches for required information on the internet.

Secondary Storage: Storage other than ROM or RAM that is non-volatile. It holds data when the power is removed and is used to store software/files not immediately needed by the computer. It is also used for backup copies.

Secure Socket Layer (SSL): A standard used for securing transactions made over the internet. SSL allows an encrypted link to be set up between two computers connected via the internet, protecting communication from being intercepted and verifying the identity of the website.

Serial/Sequential Access: Data is accessed from the storage media by starting at the beginning and continuing until the required data is found. This is a slower form of access used primarily for tape storage.

Spellchecker: A program usually found with word processors and text editing software, which checks spelling in a document and suggests correctly spelled words.

Spyware: Software installed on a computer without the owner's knowledge and consent, used to monitor the user's activity, such as tracking keystrokes to record usernames and passwords for fraudulent use.

Standalone Computer: A computer that is used on its own, without any connection (wired or wireless) to a network, including the internet.

Storage Media: The collective name for different types of storage materials such as DVDs, magnetic hard disks, solid-state memory cards, etc.

Style Sheet: A document that defines fonts and font sizes for headings, subheadings, and other elements in a document. Changes made in the style sheet are reflected across the entire document.

Swipe Card: A plastic card containing data stored on a magnetic stripe, typically used for access control or payments.

Switch: A device that inspects data packets and forwards them to the correct computer, reducing network congestion by only sending data to the intended recipient.

Social Networking Site: A website used to communicate with friends, family, and new contacts, sharing information, messages, and updates.

Software: Programs that control the operation of a computer or process electronic data.

Software License: A document (digital or paper) that sets out the terms under which software can be used, including how many computers it can be run on simultaneously.

Solid-State Backing Storage: The smallest form of memory, used as removable storage. It is robust, with no moving parts, and stores data without needing electricity. Examples include memory sticks, pen drives, and flash memory cards.

Solid-State Device: A device that uses chips and circuits, rather than spinning disks and mechanical systems, to store and retrieve data.

Sound Card: An expansion card that allows a computer to send audio signals to devices such as speakers or headphones.

Spam: Unsolicited bulk email, often from unknown senders, intended to promote goods or services, usually sent to a large audience with the hope that a small percentage may purchase the offerings.

Systems Software: Programs that control computer hardware directly by providing step-by-step instructions that tell the hardware what to do.
